## Caption 组件

### 基础用法

```tsx
import React from 'react';
import { Caption } from 'mm-lib';

export default () => <Caption className="ecology">基础信息</Caption>;
```

### API

| 属性     | 说明     | 类型      | 默认值 |
| -------- | -------- | --------- | ------ |
| children | 标题内容 | ReactNode | -      |
